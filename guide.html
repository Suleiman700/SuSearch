SuSearch

A Fuzzy JavaScript Library That Allows You To Search In A Large Amount Of Data

In order to get started with SuSearch you'll need to follow some steps, Don't worry its easy :)

jQuery

SuSearch needs jQuery in order to work, You can easily import it by using the code block below

```javascript // Import jQuery <script src="https://code.jquery.com/jquery-3.6.0.min.js" integrity="sha256-/xUj+3OJU5yExlq6GSYGSHk7tPXikynS7ogEvDej/m4=" crossorigin="anonymous"> ```

Import SuSearch Styling

SuSearch needs jQuery in order to work, You can easily import it by using the code block below

```html <!-- SuSearch Style --> <link rel="stylesheet" href="/Path/To/SuSearch/SuSearch.css"> ```

Import SuSearch

To import SuSearch script simply add this line to your .js file.

```javascript // Your script.js file import SuSearch from "/javascript/Classes/SuSearch/SuSearch.js" // SuSearch ```

Please notice that your .js type should be module, Example:

```html <!-- Your HTML file --> <html> <head> <!-- SuSearch Style --> <link rel="stylesheet" href="/Path/To/SuSearch/SuSearch.css"> </head> <body> <!-- Script file that contains SuSearch settings --> <script src="./assets/js/script.js" type="module"> </body> </html> ``` ```javascript // Your script.js file import SuSearch from "/javascript/Classes/SuSearch/SuSearch.js" // SuSearch ```

Set SuSearch Parent Div

In order for SuSearch to work, it needs a parent div

```html <!-- SuSearch parent div --> <div id="SuSearchParent"></div> ```

Example

```html <!-- Your HTML file --> <html> <head> <!-- SuSearch Style --> <link rel="stylesheet" href="/Path/To/SuSearch/SuSearch.css"> </head> <body> <!-- SuSearch parent div --> <div id="SuSearchParent"></div> <!-- Script file that contains SuSearch settings --> <script src="./assets/js/script.js" type="module"> </body> </html> ```

Please notes

  • Please note that the parent div must have a non-used id.
  • Parent div can be placed anywhere you want in your page.

Set SuSearch Data Source

SuSearch source needed to be set before doing any search.

```javascript // Usage const source = [ { "id": 1, "title": "The Beetlejuice", "year": "1988", "runtime": "92", "actors": "Alec Baldwin, Geena Davis, Annie McEnroe, Maurice Page", } ] SuSearch.set_search_source = source ```
```javascript // Example of your script.js file import SuSearch from "/javascript/Classes/SuSearch/SuSearch.js" // SuSearch const source = [ { "id": 1, "title": "The Beetlejuice", "year": "1988", "runtime": "92", "actors": "Alec Baldwin, Geena Davis, Annie McEnroe, Maurice Page", } ] SuSearch.set_search_source = source ```

SuSearch Options

SuSearch has its own default option that are used, You can manually change settings by using the following function.

All options will be described at the bottom of the page.

```javascript // Usage SuSearch.set_custom_options(_option [String], _settings [Object]) ```
```javascript /** * Example: Set callback function when user clicks on one of the dropdown items * @param _item_value {String} * @param _item_text {String} * @param _item_data {Object} */ const SuSearch_CallBackFunc = (_item_value, _item_text, _item_data) => { // _item_value => The value of the selected item // _item_text => The text of the selected item // _item_data => All the data of the selected item } SuSearch.set_custom_options('ItemsSettings', { Callback: SuSearch_CallBackFunc }) ```
```javascript // Example: Enable "Search All Results" button inside the dropdown SuSearch.set_custom_options('SearchAllSettings', { State: true }) ```

Set Having Options

Having options is important in case you need to search for items that have a specific key.

Setting having options can be done at any time, For example you can set the having option when a user selects item from select element

For example, you want to search for all the items that have isOnSale=1, and this is what options are made for.

  • Allowed operations: == === != !== > >= < <=
  • Please note that you can set multiple options
```javascript // Usage SuSearch.set_having = [ { key: 'Item key', operation: 'Your operation', value: 'Value you want' }, ] ```
```javascript // Example of your script.js file import SuSearch from "/javascript/Classes/SuSearch/SuSearch.js" // SuSearch const source = [ { "id": 1, "title": "Google Pixel 6 Pro", "isOnSale": "true", "Price": "3000", "Barcode": "123456789", } ] SuSearch.set_search_source = source // This only searches for items that have isOnSale===true and Price >= 1500 SuSearchV2_C.set_having = [ { key: 'isOnSale', operation: '===', value: 'true' }, { key: 'Price', operation: '>=', value: '1500' }, ] ```
Full example
```html <!-- Your HTML file --> <html> <body> <select class="form-control" id="filter_isOnSale"> <option value="true">True</option> <option value="false">False</option> </select> <!-- Script file that contains SuSearch settings --> <script src="./assets/js/script.js" type="module"> </body> </html> ``` ```javascript // Example of your script.js file import SuSearch from "/javascript/Classes/SuSearch/SuSearch.js" // SuSearch $('#filter_isOnSale').on('change', () => { const selected = $('#filter_isOnSale').val() // true | false // This only searches for items that have isOnSale===true SuSearchV2_C.set_having = [ { key: 'isOnSale', operation: '===', value: selected' }, ] }) ```

To initialize SuSearch, simply use this code line below:
